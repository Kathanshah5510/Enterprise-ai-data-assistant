# Enterprise AI Data Assistant

An enterprise-grade AI-powered data assistant that enables employees to query organizational databases using natural language while enforcing strict Role-Based Access Control (RBAC).

## Features

- Natural Language to SQL using Large Language Models
- LangChain + LangGraph based agent workflow
- Retrieval-Augmented Generation (RAG)
- PostgreSQL backend with Row-Level Security (RLS)
- JWT Authentication
- Role-Based Access Control (RBAC)
- Conversation Memory
- Query History
- SQL Validation & Guardrails
- Execution Metrics
- Automatic Data Visualization
- Audit Logging
- Cached Schema Embeddings
- Docker & Kubernetes Deployment

## Tech Stack

- Python
